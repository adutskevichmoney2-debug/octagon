# Октагон — кинетический этюд № 08

Физически точная симуляция: упругие мячи внутри вращающегося восьмиугольника.

**Сайт:** https://adutskevichmoney2-debug.github.io/octagon/

## Особенности

- Импульсные столкновения с учётом скорости вращающейся стены (u = ω × r)
- Соударения мячей друг с другом: массы ∝ r², обмен вращением при ударах
- Кулоново трение, порог реституции, сопротивление качению
- Физика на фиксированных подшагах 480 Гц — без туннелирования
- Один самодостаточный HTML-файл: Canvas 2D, без библиотек

## Управление

- Перетащите мяч, чтобы бросить
- Двойной клик по пустому месту — новый мяч
- Пробел — пауза · R — сброс
- Слайдеры: вращение, гравитация, упругость, трение · кнопки +/− — количество мячей
